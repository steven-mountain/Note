### TOP - "place"

Greek

topic: is a subject rather than a place

意为“**与普通地方有关的，关于地方的，当地的**”，源自 *topos* “地方”（参见 [topos](https://www.etymonline.com/cn/word/topos)）。 “**在演讲或写作中处理的事项，主题，主题**”的意思首次记录于1720年。

+ topical
  + **adj.**有关时事的；当前关注的；热门话题的；（身体）局部的
  + **网络**论题的；主题的；题目的
  + 1580年代，“**与某个地方有关的**”； 见 [topic](https://www.etymonline.com/cn/word/topic) + [-al](https://www.etymonline.com/cn/word/-al)（1）。医学上“应用于身体的特定部位”的意思始于大约1600年。“与当今话题有关的”意义始于1873年。相关： *Topically*。



+ ectopic
  + **adj.**（妊娠）异位的
  + **网络**异位性；异位电活动；异位突变体
  + occurring in an abnormal position or in an unusual manner or form




+ utopian
  + **n.**理想家；乌托邦[理想国]的居民； **网络**乌托邦式的；理想化的；空想的
  + 1550年代，指莫尔的虚构国家； 1610年代，指“荒诞不经的理想，不可能实现的幻想”，源自 [utopia](https://www.etymonline.com/cn/word/utopia) + [-an](https://www.etymonline.com/cn/word/-an)。作为一个名词，意思是“幻想家理想主义者”，记录于1832年（在这个意义上也有 *utopiast*，1845年）。***Utopian socialism*** 源于1849年，最初是贬义的，
  + 源自希腊语 *ou* “不”+ *topos* “地方”（参见 [topos](https://www.etymonline.com/cn/word/topos)）。关于希腊语 *ou* “不”的现行解释（自大约1960年起）是奇怪的，因为它将这个词源自原始印欧语 [*aiw-](https://www.etymonline.com/cn/word/*aiw-) “生命力，生命； 长寿，永恒”。
  + 通常，但不正确地被认为是源自希腊语 ***eu**-* “**好**”（参见 [eu-](https://www.etymonline.com/cn/word/eu-)），




+ topography
  + **n.**地形；测绘学
  + **网络**地形学；地势；地形图
  + 15世纪早期，“**地方描述**”，源自晚期拉丁语 *topographia*，来自希腊语 *topographia* “地方描述”，源自 *topos* “地方”（见 [topos](https://www.etymonline.com/cn/word/topos)）+ *-graphia*（见 [-graphy](https://www.etymonline.com/cn/word/-graphy)）。意思是“**地区的集体特征**”




### CENTR/CENTER - "sharp point" or "center point of a circle"

Greek kentron,

Latin centrum



+ eccentric
  + **n.**怪人；偏心圆；【机】偏心器；【天】离心圈
  + **adj.**古怪的；异乎寻常的
  + 15世纪早期，“**离心圆或轨道**”，最初是托勒密天文学中的一个术语，“**圆或轨道不是地球的中心**”，源自法语 *eccentrique*，直接来自中世纪拉丁语 *eccentricus*（名词和形容词），源自希腊语 *ekkentros* “偏离中心”（与 [concentric](https://www.etymonline.com/cn/word/concentric) 相对），源自 *ek* “出”（见 [ex-](https://www.etymonline.com/cn/word/ex-)) + *kentron* “中心”（见 [center](https://www.etymonline.com/cn/word/center)（n.））。




+ epicenter
  + **n.**震中；焦点
  + **网络**震央；中心；浩劫惊魂
  + 地震学中的1885年，“地震中心或焦点正上方的地球表面点”，源自现代拉丁语 *epicentrum*（地质学中的1879年）； 参见 [epi-](https://www.etymonline.com/cn/word/epi-) + [center](https://www.etymonline.com/cn/word/center)（n.）




+ egocentric
  + **adj.**以自我为中心的；自私自利的
  + **n.**自我主义者




+ ethnocentric
  + **adj.**种族（或民族）中心主义的；种族（或民族）优越感的
  + "本民族是文明中心"，1891年，来自 [ethno-](https://www.etymonline.com/cn/word/ethno-) + [-centric](https://www.etymonline.com/cn/word/-centric)； 社会科学中的技术术语，直到20世纪下半叶开始更广泛地使用。
  + **ethno-** : 这个词源元素的意思是“**种族，文化**”，源自希腊语 *ethnos* “**人民，国家，阶级，种姓，部落**； 一群习惯生活在一起的人”




### DOM - "house" or "master"

Latin, domus "house"

dominus "master"

+ dominion
  + **n.**支配；领土；统治（权）；管辖
  + 15世纪中叶，指“统治、至高无上的权力”，源自古法语 *dominion* “领土、统治、权力”，直接源自中世纪拉丁语 *dominionem*（主格 *dominio*），对应于拉丁语 *dominium* “财产、所有权”，源自 *dominus* “领主、主人”，源自 *domus* “房屋”




+ predominant
  + **adj.**显著的；明显的；盛行的；占优势的
  + **n.**【生】特优生物
  + **网络**主要的；突出的；占主导地位的
  + 1570年代，“**统治的； 控制的； 施加权力、权威或影响力的**”，源自法语 *prédominant*（14世纪），来自中世纪拉丁语 **praedominantem*（主格 *praedominans*）， **praedominare* 的现在分词形式，源自拉丁语 *prae* “在...之前”（见 [pre-](https://www.etymonline.com/cn/word/pre-))和 *dominari* “**统治，支配，治理**”，源自 *dominus* “主人，主人”，源自 *domus* “房子”（源自 PIE 词根 [*dem-](https://www.etymonline.com/cn/word/*dem-) “房子，家庭”）。弱化的“**更频繁，普遍**”的意义始于大约1600年。




+ domineering
  + **v.**跋扈；高耸；对…飞扬跋扈；对…盛气凌人
  + **网络**称王称霸；作威作福；擅权
  + 1580年代，“**以专制或傲慢的方式统治**”，源自荷兰语 *domineren* “统治，扮演主人； 宴请”（16世纪），来自法语 *dominer*，源自拉丁语 *dominari* “**统治，‘主宰**’”，来自 *dominus* “**主人，大师**”，字面意思是“**家庭主人**”，来自 *domus* “房子，家”（来自 PIE 词根 [*dem-](https://www.etymonline.com/cn/word/*dem-) “房子，家庭”）+ ***-nus*，表示所有权或关系的后缀**。
  + ***-nus*，表示所有权或关系的后缀**。




+ domination
  + **n.**统治；优势；【生】显性化；【宗】主天使
  + **网络**支配；控制；宰制




### OMNI - "all"

Latin omnis



+ omnivore
  + **n.**杂食动物；杂食的人
  + **adj.**杂食动物的
  + **vor-, eat**




+ omnipotent
  + **adj.**万能的；全能的；无所不能的
  + **n.**万能者；神
  + 14世纪早期，“**全能的，拥有无限力量的**”，来自古法语 *omnipotent* “**全能的，全能的**”（11世纪）和直接来自拉丁语 *omnipotentem*（格位 *omnipotens*）“**全能的，全能的**”，来自 ***omnis*** “**全部**”（见 [omni-](https://www.etymonline.com/cn/word/omni-)) + ***potens***（属格 *potentis*）“**有力的**”（见 [potent](https://www.etymonline.com/cn/word/potent)）。




+ omnibus
  + **n.**公共汽车；（广播、电视）综合节目；（若干种作品的）汇编
  + **adj.**综合性的；选编的
  + 1829年，指“**长身体、四轮的公共交通工具，有座位供乘客使用**”，源自法语 *(voiture) omnibus*，“**所有人的（马车），普通（交通工具）**”，源自拉丁语 *omnibus*，“**为所有人**”， *omnis* 的复数与格，意为“所有的”（见 [omni-](https://www.etymonline.com/cn/word/omni-)）。




+ omniscient
  + **adj.**无所不知的；全知全能的；博闻广识的
  + **n.**上帝
  + "**拥有所有知识，具有普遍知识**"，约于1600年，源自现代拉丁语 *omniscientem*（主格 *omnisciens*）"**全知的**"，是从中世纪拉丁语 *omniscientia* "全知"中反推出来的，由拉丁语 ***omnis*** "**所有**"（见 [omni-](https://www.etymonline.com/cn/word/omni-))和 ***scientia*** "**知识**"（见 [science](https://www.etymonline.com/cn/word/science)）组成。




### HOL/HOLO - "whole"

Greek, holos

+ holistic
  + **adj.**整体的；全面的；功能整体性的
  + **网络**全人；全盘的；整体论
  + 1926年，来源于 [holism](https://www.etymonline.com/cn/word/holism)（参见）+ [-istic](https://www.etymonline.com/cn/word/-istic)。*Holistic medicine* 在1960年得到确认。
  + **-istic**  :  形容词词缀，




+ hologram
  + **n.**全息图
  + **网络**全像；全息照相；全像图
  + 1949年，由匈牙利出生的英国科学家丹尼斯·加博尔（Gábor Dénes）创造，他因**全息术**的研究获得1971年诺贝尔物理学奖； 源自希腊语 *holos* “整个”（此处指“三维的”； 源自 PIE 词根 [*sol-](https://www.etymonline.com/cn/word/*sol-) “整个，保存完好”）+ [-gram](https://www.etymonline.com/cn/word/-gram)。



+ holocene
  + **adj.**【地】全新统的

  + **caustic** ：能够燃烧或破坏有机组织，腐蚀性的

  + "关注或涉及过去时期的，"1660年代，来自 [retrospect](https://www.etymonline.com/cn/word/retrospect) + [-ive](https://www.etymonline.com/cn/word/-ive)。作为名词，到1964年，缩写为 *retrospective exhibition*（1908年）




### TEMPOR - "time"

Latin tempus



+ temporal
  + **n.**颞骨[肌,动脉等]；太阳穴；世俗的权力；一时的事物
  + **adj.**世间的；世俗的；现世的；时间的
  + 14世纪晚期，“**世俗的，世间的**”； 也指“**地球的，临时的，只持续一段时间的**”，源自古法语 *temporal* “世俗的”，直接源自拉丁语 *temporalis* “**关于时间的，表示时间的**； 但只是一段时间的，临时的”，源自 ***tempus***（属格 *temporis*）“时间，季节，时刻，适当的时间或季节”，**源自原意为“拉伸，测量”的**原始意大利语 **tempos-*，根据德·瓦恩的说法，这个词源自 PIE **temp-os* “拉伸”，源自词根 [*ten-](https://www.etymonline.com/cn/word/*ten-) “拉伸”，其概念是“时间的延伸”。




+ contemporary
  + **adj.**现代的；当代的；当时的；同时代的
  + **n.**与…同时代的人；当时的人；同龄人
  + **网络**现代舞；当代艺术
  + 1630年代，“**同时发生，生活或存在于同一时期，属于同一时代或时期**”，源自中世纪拉丁语 *contemporarius*，来自拉丁语 ***com*** “**与，一起**”（见 [con-](https://www.etymonline.com/cn/word/con-)) + *temporarius* “**时间的**”，来自 *tempus* “时间，季节，一段时间”（见 [temporal](https://www.etymonline.com/cn/word/temporal)（形容词））。




+ extemporaneous
  + **adj.**不用讲稿的；脱口而出的；临时的
  + **网络**即席的；即兴；临时作成的
  + 源自拉丁短语 *ex tempore*，意为“**即兴，根据当时的需要**”，字面意思为“不合时宜”，由 *ex* “**出**”（见 [ex-](https://www.etymonline.com/cn/word/ex-))和 *tempore* 构成，




+ temporize
  + **v.**拖延时间（不迅速作决定等）
  + **网络**见风使舵；顺应时势；妥协
  + "**顺应时代潮流**； **在观点或环境的影响下表面上顺从**", 1550年代(暗示于 *temporizer* ), 源自法语的 *temporiser* "度过自己的时间，等待自己的时间" (14世纪), 源自于中世纪拉丁语的 *temporizare* "**消磨时间**"，或许是通过俗拉丁语的 **temporare* "**耽搁**"，来自于拉丁语的 *tempus* (属格 *temporis* ) "时间" 




### CHRON - "time"

Greek

+ chronic
  + **adj.**长期的；慢性的；难以治愈（或根除）的；长期患病的
  + **n.**慢性病人
  + 15世纪初， *cronik*，用于**描述疾病**，意为“**持续很长时间**”，源自古法语 *chronique*，直接源自拉丁语 *chronicus*，来自希腊语 *khronikos*，意为“**关于时间的**”，源自 *khronos* “时间”（参见 [chrono-](https://www.etymonline.com/cn/word/chrono-)）。17世纪以来，**由于与疾病和后来的瘾症有关**，产生了**模糊的贬义**意义。在英语中，字面意义上的“与时间有关”的用法很少见。




+ chronology
  + **n.**按事件发生的年代排列的顺序；年表
  + **网络**年代学；年代表；时序
  + 1590年代，“**时间科学**”，源自法语 *chronologie* 或直接源自现代拉丁语 *chronologia*； 参见 [chrono-](https://www.etymonline.com/cn/word/chrono-) + [-logy](https://www.etymonline.com/cn/word/-logy)。相关词汇： *Chronologer*（1570年代）。指“**某些过去事件的假定顺序的特定陈述**”始于1610年代。




+ anachronism
  + **n.**过时的人（或风俗、思想）；弄错年代；时代错误
  + **网络**不合时宜的；年代错误；倒错
  + 1640年代，“计算时间或查找日期时的错误”，源自拉丁语 *anachronismus*，来自希腊语 *anakhronismos*，源自 *anakhronizein* “参照错误的时间”，由 ***ana*** “**反对**”（见 [ana-](https://www.etymonline.com/cn/word/ana-))和 *khronos* “**时间**”（见 [chrono-](https://www.etymonline.com/cn/word/chrono-))组成。
  + **ana-** ：元音前的词缀 *an-*，意为：1. "**向上，在地点或时间上向上**"，2. "**向后，反对**"，3. "**再次，重新**"




+ synchronous
  + **adj.**同时发生（或存在）的；同步的；共时的
  + **网络**同时的；同期的；同时发生的
  + 1660年代，“同时存在或发生的”，源自晚期拉丁语 *synchronus* “同时的”，来自希腊语 *synkhronos* “同时发生的”，由 *syn-* “**一起**”（见 [syn-](https://www.etymonline.com/cn/word/syn-))和 *khronos* “时间”（见 [chrono-](https://www.etymonline.com/cn/word/chrono-))组成。意思是“**在同一时间的连续瞬间重复出现**”
