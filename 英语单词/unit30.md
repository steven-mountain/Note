### idio

和自己有关系，和个人有关系。自我

+ idiom
  + n 习语
  + 属于特定的人的
+  idiomatic
  + adj 自然地道的
+ idiosyncratic
  + adj 独特的
  +  从这个表格可以看出，“idiosyncratic” 的字面意思是“**个人特有的混合体**”，引申为“**特质的、独特的、异质的**”。
+ idiopathic
  + adj 自发的，先天的，特发的
  + "Idiopathic" 源自希腊语 "**idios**"（自身的）和 "**pathos**"（痛苦、疾病）的组合。
  + 其本义是“**自身发生的疾病**”，即**找不到外在原因（如感染、创伤、中毒等），而是源于身体内部自身功能的紊乱或未知因素**的疾病。
+ idiot
  + n 蠢人
